#Proyecto obesidad y otras enfermedades crónicas en la ciudad de Córdoba
Les presentamos nuestra base de datos.. Es un dataset de base poblacional, con datos reales de 4300 sujetos del gran Córdoba que contiene información sobre las características sociodemográficas, antropométricas, de enfermedades crónicas no transmisibles (obesidad, cáncer, enfermedad cardiovascular, diabetes, entre otras), y del estilo de vida, incluyendo en este último aspecto información sobre alimentación y nivel de actividad física. Hay disponible más de 100 features.
